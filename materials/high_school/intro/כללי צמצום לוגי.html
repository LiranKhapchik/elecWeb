אלגברת מיתוג

אקסיומה: קיים משתנה מיתוג בעל שני ערכים 0 וְ- 1.
$$ x=1 ↔ x≠0 $$ $$ x=0 ↔ x≠1 $$

אקסיומה אלגברת מיתוג: מערכת אלגברית של הקבוצה {1 , 0} המכילה שתי בפועלות בינאריות (פעולות על שני משתנים או יותר) ופעולה אונרית (פעולה על משתנה אחד) אחת, בהמשך נראה שאפשר לשלב בין פעולות.
  • OR לוגי: תוצאתה "אמת" כאשר אחד האופרנדים שלה (או שניהם), הוא "אמת", וביתר המקרים תוצאתה "שקר".
  • AND לוגי: תוצאתה "אמת" כאשר שני האופרנדים שלה הם "אמת", וביתר המקרים תוצאתה "שקר".
  • NOR לוגי: תוצאתה "אמת" כאשר שני האופרנדים שלה הם "שקר", וביתר המקרים תוצאתה "שקר".

הוכחה באינדוקציה שלמה

לאלגברה בוליאנית 7 תכונות בסיסיות שאותן נוכח בעזרת הוכחה באינדוקציה שלמה 
שהיא שיטה להוכחת תכונות משופות המורכבת משני שלבים:

  • שלב 1: הוכחה של מספר 1 (N).
  • שלב 2: שלב האינדוקציה: מוכיחים עבור N ועבור  N + 1.

    • בגלל שהקבוצה הבוליאנית מורכבת רק משני סימנים {1 , 0} , אפשר להוכיח הוכחה שלמה.

    תכונות בסיסיות

    1 (א'). אדישות

  • $x + x = x$
  • $x · x = x$
  • $0 + 0 = 0$
  • $1 + 1 = 1$
  • $0 · 0 = 0$
  • $1 · 1 = 1$

    • 1 (ב'). ערכים אדישים ושולטים

  • 1 דומיננטי ב -   OR : $ x + 1 = 1$
  • 0 דומיננטי ב  -AND : $ x · 0 = 0$
  • 0 אדיש ב  -OR  : $ x + 0 = x$
  • 1 אדיש ב  - AND : $ x · 1 = x$

    • 2. השלמה - ערך משלים לאדיש:

    ערך משלים של משתנה מתבטא כך:

  • $x +\bar{x}= 1$
  • $x · \bar{x} = 0$
    • (  $\bar{x}$ מייצג לי not(x)  )

    3. קוגניטיביות חילופית (חוק החילוף):

    מאפשרים להחליף בין האיברים בלי לשנות את התוצאה.

  • $x + y = y + x$
  • $x · y = y · x$

    • 4. אסוציאטיביות קיבוצית (חוק הקיבוץ):

    מאפשרים להחליף בין האיברים בלי לשנות את התוצאה.

  • $ (x + y) + z = x + (y + z)$
  • $ (x · y) · z = x · (y · z)$

    • 5. דיסטרביוטיות פילוג (חוק הפילוג):

    מאפשר לפזר או לאחד איברים בין פעולות לוגיות שונות.

  • $ x · (y + z) = x · y + x · z$
  • $ x + y · z = (x + y) · (x + z)$

    • 6 (א'). עקרון הבליעה (Absorption Law):

    מציין שהוספת איבר מסוים לתוצאה של פעולה על המשתנה אינה משנה את הערך.
    6(א.)

  • $x + x · y = x$
  • $x · (x + y) = x$
    • 6(ב.)
  • $x + \bar{x}· y = x + y$
  • $x · (\bar{x} + y) = x · y$

    • 7. כלל דה מורגן:

    מאפשר לפזר או לאחד איברים בין פעולות לוגיות שונות.

  • $ \overline{( x ∙ y )} =  \bar{x} +  \bar{y} $
  • $  \overline{( x + y )} =  \bar{x} ∙  \bar{y}$

    • עיקרון הדואליות

    עקרון הדואליות באלגברה בוליאנית קובע שלכל זהות אלגברית בוליאנית קיימת זהות נוספת, הנקראת הדוּאָלית שלה, אשר מתקבלת על־ידי החלפה עקבית של:

    • פעולת AND (⋅) ב־OR (+)
    • פעולת OR (+) ב־AND (⋅)
    • הערך 0 ב־1, והערך 1 ב־0
    התוצאה היא זהות תקפה נוספת באלגברה הבוליאנית.
    דוגמא #1 : נזכר בעיקרון 1ב' (ערכים עדישים ושולטים)
    זהות לערך אדיש: $A + 0 = A$
    זהות הדוּאָלית שלה: $A ⋅ 1 = A$
    זהות לערך שולט: $A + 1 = 1$
    זהות הדוּאָלית שלה: $A ⋅ 0 = 0$

    דוגמא #2 : לפי דיסטריבוטיביות פילוג (חוק הפילוג)
    הנוסחה המקורית: $A ⋅ (B + C) = A⋅B + A⋅C$
    הדוּאָל של החוק (על פי עקרון הדואליות): $A + (B ⋅ C) = (A + B) ⋅ (A + C)$
    שימו לב שתכונות הזהויות נשארו זהות לפי ההגדרה שלנו
    הדואליות לא אומרת שהנוסחה "שווה" למשהו אחר, אלא שיש לה גרסה הפוכה (עם הפעולות ההפוכות), שגם היא זהות נכונה.

    פעולה זו אינה מורשית.
